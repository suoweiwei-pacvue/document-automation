# 分享与协作

## 功能简介

Custom Dashboard 提供了两大协作能力，帮助你将数据看板和图表模板高效地传递给团队内外的相关人员：

- **ShareLink 外部分享**：将你创建好的 Dashboard 生成一个链接，发送给客户、合作伙伴或其他外部人员。对方无需注册或登录 Pacvue 平台，打开链接即可查看看板数据。你还可以为链接设置访问密码和锁定数据的时间范围，确保信息安全可控。
- **模板跨 Client 共享**：如果你管理多个 Client（客户账户），可以将常用的图表模板分享给其他 Client 使用，避免在不同账户下重复创建相同配置的图表，特别适合代理机构（Agency）和总部（HQ）场景。

## 使用场景

- **向客户汇报广告表现**：作为代理机构，你需要定期向客户展示广告投放效果。通过 ShareLink 生成链接并发送给客户，客户打开即可查看，无需额外授权。
- **锁定数据时间范围后分享**：你想分享 2 月份的投放数据给客户，但不希望客户自行切换到其他月份查看。通过锁定时间范围功能，客户只能看到你指定时段的数据。
- **保护敏感数据**：分享的看板包含敏感的业务数据，你可以为链接设置访问密码，只有知道密码的人才能查看。
- **跨账户复用图表模板**：你为某个 Client 精心配置了一套图表模板，希望在其他 Client 账户下也能直接使用，无需从头配置。通过模板分享功能，一键将模板共享给目标 Client。
- **团队协作统一看板标准**：总部希望所有子账户使用统一的图表模板来生成报告，通过批量分享模板，快速实现标准化。

## 操作指南

### 一、ShareLink 外部分享

#### 1.1 创建分享链接

通过分享链接，你可以让外部人员在不登录 Pacvue 的情况下查看你的 Dashboard。

**前提条件**：Dashboard 必须已保存（非草稿状态）。处于草稿（Draft）状态的 Dashboard 不会显示分享图标。

**操作步骤**：

1. 进入 Custom Dashboard 列表页。
2. 将鼠标悬浮到你想分享的 Dashboard 卡片上，卡片上会出现操作图标。
3. 点击 **分享图标**。

[截图: Dashboard 列表页，鼠标悬浮在某个 Dashboard 卡片上，标注分享图标的位置]

4. 系统会自动生成一个分享链接，并**立即复制到你的剪贴板**。
5. 同时弹出 **"Copy Share Link"** 弹窗，你可以在弹窗中进一步设置密码和时间锁定（详见下文）。
6. 将剪贴板中的链接通过邮件、即时通讯工具等方式发送给需要查看的人。

[截图: Copy Share Link 弹窗界面]

> **注意事项**：
> - 分享链接是动态链接——当你对 Dashboard 进行修改并保存后，通过分享链接看到的内容也会随之更新。
> - 分享图标仅在 Dashboard 列表页显示，在 Dashboard 的查看页面（Display 页面）和编辑页面中没有分享入口。
> - 每次点击分享图标都会将最新的链接复制到剪贴板，你可以随时重新复制。

---

#### 1.2 设置、修改或取消访问密码

为分享链接设置密码后，外部人员必须输入正确的密码才能查看 Dashboard 内容。

**设置密码**：

1. 点击分享图标后，在弹出的 **"Copy Share Link"** 弹窗中，开启密码保护开关。
2. 在密码输入框中输入你想设置的密码。
3. 点击 **保存** 按钮。
4. 密码设置成功后，外部人员访问链接时将被要求输入密码。

[截图: 弹窗中密码设置区域，标注密码开关和输入框]

**修改密码**：

1. 再次点击该 Dashboard 的分享图标，弹窗会显示当前已设置的密码。
2. 修改密码内容。
3. 点击 **保存** 按钮完成修改。

**取消密码**：

1. 再次点击该 Dashboard 的分享图标。
2. 关闭密码保护开关。
3. 点击 **保存** 按钮。
4. 取消密码后，任何拥有链接的人都可以直接查看 Dashboard，无需输入密码。

> **注意事项**：
> - 每个 Dashboard 的分享链接只能设置一个密码。
> - 密码在系统中以加密方式存储，确保安全性。
> - 修改或取消密码后立即生效，已拿到链接的人下次访问时将按新的密码规则验证。

#### 1.3 锁定数据时间范围

默认情况下，分享链接是动态的——如果你在 Dashboard 上调整了时间范围，外部人员看到的数据时间范围也会跟着变化。**锁定时间范围**功能可以解决这个问题，让外部人员只能查看你指定时段的数据。

**典型场景**：
- 你想分享 2 月的广告数据给客户，但不希望客户自行切换到 1 月或 3 月查看。
- 你正在调整 Dashboard 的时间范围做内部分析，不想影响客户通过分享链接看到的数据。

**操作步骤**：

1. 点击分享图标，打开 **"Copy Share Link"** 弹窗。
2. 开启 **锁定时间范围** 开关。
3. 设置时间范围：
   - 选择 **日期范围类型**：可以选择"自定义（Custom）"或系统预设的固定日期范围（如"上个月""最近 7 天"等）。
   - 如果选择自定义，手动设置 **开始日期** 和 **结束日期**。
4. 点击 **保存** 按钮。

[截图: 弹窗中时间锁定设置区域，标注开关、日期范围类型下拉框和日期选择器]

5. 锁定后，外部人员访问分享链接时，页面上的时间选择器将被禁用，只能查看你指定时段的数据。

**取消锁定**：

1. 再次打开分享弹窗。
2. 关闭 **锁定时间范围** 开关。
3. 点击 **保存**。取消后，外部人员可以自行调整时间范围。

> **注意事项**：
> - 锁定时间范围功能支持所有广告平台（Amazon、Walmart、DSP 等）。
> - 只有开启了锁定开关，设置的开始和结束日期才会生效。如果只填了日期但没有开启开关，时间范围不会被锁定。

#### 1.4 外部人员如何访问分享链接

当外部人员（如你的客户或合作伙伴）收到分享链接后，访问流程如下：

1. 点击或在浏览器中打开分享链接。
2. 系统自动判断该链接是否需要密码：
   - **不需要密码**：直接进入 Dashboard 查看页面，可以看到所有图表和数据。
   - **需要密码**：页面弹出密码输入框，输入正确密码后进入查看页面。

[截图: 外部访问时的密码输入页面]

3. 进入查看页面后，外部人员可以看到与你在 Dashboard 查看模式下完全一致的内容，包括：
   - 顶部概览数据
   - 各类图表（趋势图、柱状图、饼图、表格、对比图等）
   - 数据提示信息（Brief Tips）

**分享页面与正常查看页面的区别**：

| 功能 | 正常查看页面 | 分享链接页面 |
|---|---|---|
| 查看图表和数据 | ✅ | ✅ |
| 面包屑导航 | ✅ 显示 | ❌ 不显示 |
| "Edit Dashboard" 按钮 | ✅ 显示 | ❌ 不显示 |
| 调整时间范围 | ✅ 可调整 | 取决于是否锁定时间 |
| 需要登录 Pacvue | ✅ 需要 | ❌ 不需要 |

**密码输入错误或链接失效的情况**：

- **密码错误**：页面提示 "Password is wrong."，可以重新输入。
- **链接失效**（Dashboard 已被删除）：页面显示缺省提示页面，说明该链接已不可用。

> **注意事项**：
> - 分享页面展示的数据内容以 Dashboard 中各图表的配置（Chart Setting）为准，所见即所得。
> - 外部人员无法对 Dashboard 进行任何编辑操作，只能查看。

### 二、模板跨 Client 共享

#### 2.1 功能说明

如果你在 Pacvue 平台上管理多个 Client（客户账户），模板共享功能可以让你将一个 Client 下创建的图表模板分享给其他 Client 使用。这在以下场景中特别有用：

- **代理机构管理多个品牌**：你为品牌 A 创建了一套效果很好的图表模板，希望品牌 B 也能直接使用。
- **总部统一标准**：总部（HQ）希望所有子账户使用统一的图表模板来生成报告。

> **说明**：模板跨 Client 共享功能主要适用于 HQ（总部）场景。

#### 2.2 分享模板给其他 Client

**操作步骤**：

1. 进入 **Chart Library**（图表模板库）页面。在 Custom Dashboard 首页点击 **【Chart Library】** 按钮即可进入。

[截图: Custom Dashboard 首页，标注 Chart Library 按钮位置]

2. 在模板管理页面，勾选你想要分享的一个或多个模板。
   - 你可以通过左侧的分组列表、图表类型筛选、关键词搜索等方式快速找到目标模板。
   - 勾选模板后，页面底部会显示已选中的模板名称。

[截图: 模板管理页面，标注模板卡片上的勾选框和底部已选模板栏]

3. 点击顶部操作栏的 **【Bulk Operation】** 按钮，在下拉菜单中选择 **【Bulk Share】**。

[截图: Bulk Operation 下拉菜单，标注 Bulk Share 选项]

4. 在弹出的分享设置弹窗中：
   - **选择目标 Client**：可以选择一个或多个 Client 作为分享对象。

[截图: 分享设置弹窗，标注 Client 选择区域和 Group 设置区域]

5. 确认设置后，点击 **分享** 按钮完成操作。

6. 分享完成后，目标 Client 的用户在其 Chart Library 中即可看到被分享的模板，可以直接引用到自己的 Dashboard 中。

## 各平台差异说明

ShareLink 外部分享功能和模板管理功能在不同广告平台上的表现基本一致，以下是需要注意的差异：

| 功能 | Amazon / Walmart / DSP | Commerce |
|---|---|---|
| 创建分享链接 | ✅ 支持 | ✅ 支持 |
| 密码保护 | ✅ 支持 | ✅ 支持 |
| 锁定时间范围 | ✅ 支持 | ✅ 支持 |
| 模板创建时的数据源选择 | 直接选择图表类型和指标 | 需要先选择 **Channel**（如 Seller、Vendor），再根据 Channel 选择对应的 Distributor View、Program、Dimension 等 |
| 模板标签自动生成 | 基于物料层级、时间维度等 | 额外包含 Data Source 信息（如 "Seller \| SnS"、"Vendor \| Sourcing \| Retail"） |
| Quick Create（从模板快速创建 Dashboard） | ✅ 支持（HQ 场景） | ❌ 不支持 |
| 模板跨 Client 批量分享 | ✅ 支持（HQ 场景） | ✅ 支持（HQ 场景） |

> **说明**：Commerce 场景下创建模板时，系统会先弹出 Channel 选择框，选择后才会展示对应的配置选项。这是因为 Commerce 平台的数据源结构更为复杂，需要先确定数据来源再进行图表配置。

## 常见问题

**Q: 分享链接发出去后，我修改了 Dashboard 的内容，对方看到的会变吗？**

A: 会的。分享链接是动态链接，你对 Dashboard 所做的修改保存后，对方通过链接看到的内容也会同步更新。如果你不希望对方看到修改后的内容，建议在修改前先锁定时间范围，或者在修改完成后再重新分享。

---

**Q: 我设置了密码，但想让某些人不用输密码就能看，怎么办？**

A: 目前每个 Dashboard 的分享链接只支持设置一个统一的密码策略——要么所有人都需要输入密码，要么所有人都不需要。无法针对不同的人设置不同的访问权限。如果你希望部分人免密访问，可以考虑取消密码设置，改为通过其他方式（如仅将链接发送给信任的人）来控制访问范围。

---

**Q: 外部人员打开分享链接后看到的是空白页面或错误提示，怎么办？**

A: 请检查以下几种可能的原因：
- **Dashboard 已被删除**：如果你删除了被分享的 Dashboard，链接将失效，对方会看到缺省提示页面。
- **链接不完整**：确认发送的链接没有被截断，完整复制并发送。
- **密码错误**：如果设置了密码，确认对方输入的密码正确。密码错误时页面会提示 "Password is wrong."。

---

**Q: 草稿状态的 Dashboard 为什么不能分享？**

A: 草稿（Draft）状态表示 Dashboard 尚未完成配置或保存。为了确保分享出去的内容是完整且准确的，系统要求 Dashboard 必须先保存为正式状态后才能生成分享链接。请先完成编辑并点击 **Save** 保存，然后再进行分享。

---

**Q: 模板分享给其他 Client 后，对方修改了模板，会影响我这边的原始模板吗？**

A: 不会。模板分享后，目标 Client 获得的是模板的一份副本。对方对副本所做的任何修改（包括重命名、调整配置、删除等）都不会影响你原始的模板。同样，你后续对原始模板的修改也不会同步到已分享出去的副本。

---

**Q: 为什么我在 Bulk Operation 中看不到 Bulk Share 选项？**

A: Bulk Share（批量分享）功能仅在 HQ（总部）场景下可用。如果你的账户不属于 HQ 角色，或者没有相应的操作权限，该选项不会显示。请联系你的管理员确认账户权限设置。
