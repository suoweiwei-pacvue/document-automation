# Dashboard 管理

## 功能简介

Dashboard 管理是 Pacvue Custom Dashboard 的核心功能模块，帮助你创建、编辑和管理个性化的数据看板。你可以在 Dashboard 中添加多种类型的图表（Chart），将广告投放、销售表现等关键数据以可视化的方式呈现出来，也可以将看板通过链接分享给团队成员或客户查看。

通过 Dashboard 管理，你可以从零开始搭建看板，也可以利用已有的图表模板快速生成看板，大幅减少重复配置的工作量。

## 使用场景

- **搭建周报/月报看板**：创建 Dashboard 并添加趋势图、对比图、汇总表格等，用于定期汇报广告投放效果。
- **分享数据给客户**：将配置好的 Dashboard 生成分享链接，客户无需登录即可查看数据，还可以设置密码保护和时间范围锁定。
- **利用模板快速建看板**：从图表模板库中选取常用的图表配置，批量应用到新 Dashboard，避免每次从头配置。
- **多平台数据整合展示**：在同一个 Dashboard 中展示 Amazon、Walmart、DSP 等不同平台的广告数据，便于横向对比。
- **团队协作与模板复用**：将常用的图表配置保存为模板，团队成员可以直接引用，保持数据口径一致。

## 操作指南

### 一、创建 Dashboard

#### 功能说明

创建一个全新的 Dashboard，作为承载各类图表的画布。创建过程中你可以添加图表、配置全局筛选器，并选择保存为草稿或正式发布。

#### 操作步骤

1. 在 Dashboard 列表页，点击 **创建** 按钮，进入 Dashboard 编辑页面。
2. 页面顶部会显示导航栏，包含以下操作按钮：
   - **Back**：返回 Dashboard 列表页（未保存的更改会丢失）。
   - **Save as Draft**：将当前 Dashboard 保存为草稿状态，草稿不会对外展示。
   - **Save**：正式保存并发布 Dashboard。
3. 在页面右上角，你可以设置 **Currency**（货币单位，如 US Dollar），所有图表中的金额类指标将以该货币展示。
4. 点击 **Dashboard Setting** 下拉菜单，可以配置 Dashboard 级别的全局筛选器（详见下文"配置全局筛选器"章节）。
5. 通过以下两种方式向 Dashboard 添加图表：
   - **Add Chart**：创建一个空白图表，手动选择图表类型并配置。
   - **Use Template**：从图表模板库中选择已有模板，快速添加预配置好的图表。
6. 添加完所有需要的图表后，点击 **Save** 正式保存，或点击 **Save as Draft** 暂存为草稿。

[截图: Dashboard 编辑页面顶部操作栏，标注 Back、Save as Draft、Save 按钮位置]

[截图: Dashboard 编辑页面主体区域，标注 Add Chart 和 Use Template 按钮]

#### 注意事项

- 草稿状态的 Dashboard 不支持生成分享链接，需要正式保存后才能分享。
- 离开编辑页面前请确保已保存，未保存的更改不会自动保留。

### 二、编辑 Dashboard

#### 功能说明

对已有的 Dashboard 进行修改，包括添加、删除、复制图表，修改图表配置，以及调整图表的布局和排列顺序。

#### 操作步骤

1. 在 Dashboard 列表页，找到目标 Dashboard，点击进入编辑模式。
2. 在编辑页面中，你可以进行以下操作：
   - **添加图表**：点击 **Add Chart** 或 **Use Template** 添加新图表。
   - **编辑图表**：点击图表上的编辑图标，打开该图表的设置弹窗，修改配置后保存。
   - **复制图表**：点击图表上的复制图标，系统会创建一份相同配置的图表副本。
   - **删除图表**：点击图表上的删除图标，确认后移除该图表。
   - **调整布局**：通过拖拽调整图表在 Dashboard 中的位置和排列顺序。
3. 修改完成后，点击 **Save** 保存更改，或点击 **Save as Draft** 保存为草稿。

### 三、删除 Dashboard

#### 功能说明

删除不再需要的 Dashboard。删除后，该 Dashboard 的分享链接也将失效。

#### 操作步骤

1. 在 Dashboard 列表页，找到目标 Dashboard。
2. 点击对应的删除操作按钮。
3. 在弹出的确认弹窗中，确认删除。

#### 注意事项

- 删除操作不可撤销，请谨慎操作。
- 如果该 Dashboard 已生成分享链接，删除后外部用户访问链接时将看到"页面不存在"的提示。

### 四、图表（Chart）类型说明

Dashboard 支持多种图表类型，每种类型适用于不同的数据展示场景：

| 图表类型 | 适用场景 | 说明 |
|---|---|---|
| **Top Overview** | 关键指标一览 | 以卡片形式展示核心指标数值，支持同比（YOY）和环比（POP）对比 |
| **Trend Chart** | 数据趋势分析 | 以折线图或柱状图展示指标随时间的变化趋势，支持按日/周/月切换展示粒度 |
| **Comparison Chart** | 多维度对比 | 对比不同物料（如不同广告活动）或不同时间段的指标表现，支持汇总对比、同比、环比三种模式 |
| **Stacked Bar Chart** | 构成分析 | 堆叠柱状图，展示各部分在整体中的占比（目前仅 Amazon 平台支持） |
| **Pie Chart** | 占比分析 | 饼图，展示单个指标在不同维度下的占比分布，每次只能选择一个指标 |
| **Table** | 明细数据查看 | 表格形式展示多指标明细数据，支持半屏/全屏显示、二级展开、添加环比数据列 |
| **Grid Table** | 网格数据展示 | 网格表格形式展示数据 |
| **White Board** | 自由内容展示 | 白板，支持插入图片，适合添加说明文字或品牌素材 |

### 五、配置图表

#### 5.1 通用配置项

所有图表类型都包含以下基础配置：

- **图表名称**：为图表起一个描述性的名称，便于识别。
- **时间范围**：设置图表查询的数据时间段。
- **物料层级（Material Level）**：选择数据的聚合维度，决定图表按什么粒度展示数据。

**物料层级选项说明：**

| 物料层级 | 含义 |
|---|---|
| Campaign | 按广告活动维度聚合 |
| Campaign Tag / Campaign Parent Tag | 按广告活动标签或父标签维度聚合，适合按业务分类查看 |
| ASIN / ASIN Parent Tag | 按商品或商品父标签维度聚合 |
| Profile | 按账户维度聚合 |
| Advertiser | 按广告主维度聚合（DSP 平台适用） |
| Campaign Type | 按广告类型维度聚合（如 SP、SB、SD） |
| Placement | 按展示位置维度聚合 |
| Cross Retailer | 跨零售商维度聚合 |
| Keyword | 按关键词维度聚合 |
| SOV Group | 按 SOV 分组维度聚合（需开通相应权限） |

#### 5.2 Trend Chart（趋势图）配置

1. 点击 Add Chart 并选择 **Trend Chart**，或点击已有趋势图的编辑图标。
2. 在设置弹窗中配置：
   - **Basic Setting**：填写图表名称、选择时间范围等基础信息。
   - **指标（Metric）选择**：选择要展示的指标（如 Spend、Sales、ACOS 等），每个指标可以独立设置展示样式为折线或柱状。
   - **数据源渠道**（Commerce 场景）：选择 Vendor（供应商）、Seller（卖家）或 Hybrid（混合）。
3. 保存后，图表展示区域支持通过 **D / W / M** 快捷按钮切换日（Day）、周（Week）、月（Month）的展示粒度。

[截图: Trend Chart 设置弹窗，标注 Basic Setting、Metric 选择、D/W/M 切换按钮]

#### 5.3 Comparison Chart（对比图）配置

Comparison Chart 支持三种对比模式：

- **BySum（汇总对比）**：按不同物料（如不同广告活动）汇总后进行对比。
- **YOY（同比）**：将当前时间段的数据与去年同期进行对比。支持多物料对比模式和多指标对比模式，也支持自定义对比时间段（Multi Periods 模式）。
- **POP（环比）**：将当前时间段的数据与上一个时间段进行对比，同样支持多种子模式和自定义对比时间范围。

配置步骤：

1. 点击 Add Chart 并选择 **Comparison Chart**。
2. 选择对比模式（BySum / YOY / POP）。
3. 选择要对比的指标和物料范围。
4. 如需自定义对比时间范围，在设置中指定对比的起止日期。
5. 保存配置。

#### 5.4 Table（表格）配置

1. 点击 Add Chart 并选择 **Table**。
2. 配置表格的显示样式：
   - **半屏 / 全屏**：选择表格占据 Dashboard 的一半宽度还是整行宽度。
3. 选择物料层级和要展示的指标列。
4. 可选配置：
   - **添加 POP 数据列**：在指标列旁边增加环比数据列，直观展示数据变化。
   - **Campaign Tag 二级展开**：当物料层级选择 Campaign Tag 时，支持展开查看父 Tag 下的汇总行。
5. 保存配置。

[截图: Table 设置弹窗，标注半屏/全屏切换和 POP 数据列选项]

#### 5.5 White Board（白板）配置

1. 点击 Add Chart 并选择 **White Board**。
2. 在白板编辑区域中插入图片或添加说明内容。
3. 保存配置。

> **提示**：白板适合用来添加报告封面、品牌 Logo、文字说明等非数据类内容，让 Dashboard 更加完整和专业。

### 六、配置全局筛选器

#### 功能说明

Dashboard 级别的全局筛选器会作用于看板中的所有图表，帮助你快速聚焦到特定范围的数据。

#### 操作步骤

1. 在 Dashboard 编辑页面，点击 **Dashboard Setting** 下拉菜单。
2. 根据需要配置以下筛选器：
   - **Advertiser Filter**（广告主筛选）：筛选特定广告主的数据，适用于 DSP 平台。
   - **Campaign Tag Filter**（广告活动标签筛选）：按标签筛选广告活动数据，适用于 Amazon 平台。
   - **Market Filter**（市场筛选）：按市场/站点筛选数据。
3. 设置完成后，Dashboard 中所有图表的数据都会按照筛选条件进行过滤。

#### 注意事项

- Advertiser Filter 仅在 DSP 平台下生效。
- Campaign Tag Filter 仅在 Amazon 平台下生效。
- 全局筛选器与图表自身的筛选条件是叠加关系，两者同时生效。

### 七、图表下载

#### 功能说明

你可以将 Dashboard 中的图表导出为 Excel 文件或图片，方便在报告、邮件中使用。

#### 操作步骤

1. 在 Dashboard 查看页面，找到目标图表。
2. 点击图表上的 **下载** 按钮。
3. 选择导出格式：
   - **下载为 Excel**：将图表数据以表格形式导出为 Excel 文件，格式根据图表类型自动适配。
   - **下载为图片**：导出纯图表图片（无背景色、无标题），适合直接插入到 PPT 或文档中。

### 八、使用图表模板

#### 8.1 进入模板管理页面

1. 在 Custom Dashboard 首页，点击 **Chart Library** 按钮，进入模板管理页面。

[截图: Custom Dashboard 首页的 Chart Library 入口按钮]

#### 8.2 浏览和筛选模板

模板管理页面分为三个区域：

- **左侧分组列表**：展示所有模板分组，点击某个分组可查看该分组下的模板。选择 "All" 查看全部模板。
- **右侧模板卡片区**：以卡片形式展示模板，每张卡片显示模板名称、图表类型、标签、创建者、引用次数（Popularity）和收藏状态。
- **顶部操作栏**：包含创建模板、快速创建 Dashboard、批量操作、搜索和筛选功能。

**筛选方式：**

- **按分组筛选**：点击左侧分组列表中的分组名称。
- **按图表类型筛选**：在顶部筛选区选择一个或多个图表类型。
- **按数据源筛选**（Commerce 场景）：选择 Seller 或 Vendor。
- **仅看收藏**：勾选 **Favorite only**，只展示你收藏过的模板。
- **关键词搜索**：在搜索框中输入模板名称关键词。

[截图: 模板管理页面整体布局，标注左侧分组、右侧卡片区、顶部筛选栏]

#### 8.3 创建模板

1. 在模板管理页面，点击 **Create Template** 按钮。
2. 在下拉菜单中选择要创建的图表类型（不支持选择 White Board）。
3. 如果是 Commerce 场景，先选择 **Channel**（渠道），系统会根据选择展示对应的配置项。
4. 填写模板基本信息：
   - **Template Name**（必填）：模板名称，在整个账户内不能重复。
   - **Group**（必填）：选择模板所属分组（不能选择系统默认分组 "Pacvue Default"）。
   - **Introduction**（选填）：模板描述说明。
   - **标签**（选填）：最多添加 3 个标签，便于分类和筛选。
5. 配置图表参数：选择物料层级、指标等（与创建图表时的配置类似，但不需要选择具体的数据范围）。
6. 右侧会实时展示假数据预览，帮助你直观了解图表效果。
7. 点击保存完成模板创建。

**预览数据说明：**
- 趋势图默认展示 5 条数据线。
- 对比图默认展示 5 组数据。
- 饼图默认展示 10 个部分。
- 表格默认展示 5-6 行数据。

[截图: 创建模板页面，标注模板名称、分组选择、图表配置区域和右侧预览区]

#### 8.4 编辑、复制和删除模板

- **编辑**：鼠标悬浮在模板卡片上，点击 **Edit** 图标，进入编辑页面修改配置。
- **复制**：点击 **Copy** 图标，系统会创建一份模板副本，你可以在副本基础上修改。
- **删除**：点击 **Delete** 图标，在确认弹窗中确认删除。

[截图: 模板卡片悬浮状态，标注 Edit、Copy、Delete、Favorite 四个操作图标]

#### 8.5 收藏模板

- 点击模板卡片上的 **星形图标**（空心）即可收藏该模板，图标变为实心。
- 再次点击实心星形图标可取消收藏。
- 收藏后，可以通过 **Favorite only** 筛选快速找到常用模板。

#### 8.6 管理模板分组

**新建分组：**
1. 在左侧分组列表旁，点击 **加号（+）** 图标。
2. 在弹窗中输入分组名称（同一账户下不能重复）。
3. 点击确认创建。

**重命名分组：**
1. 在分组名称上点击编辑操作。
2. 修改名称后保存。

**删除分组：**
1. 点击分组的删除按钮。
2. 如果该分组下有模板，系统会提示你选择：
   - 将模板移动到其他分组。
   - 连同模板一起删除。
3. 确认后执行删除。

#### 8.7 在 Dashboard 中引用模板

**单个引用：**

1. 在 Dashboard 编辑页面，点击 **Use Template** 按钮。
2. 在弹出的模板选择弹窗中，浏览或搜索模板。
   - 系统会自动过滤掉与当前 Dashboard 数据源不兼容的模板。
   - 弹窗底部会提示："The current data source is xxx. Some templates that are not applicable to this data source have been filtered out."
3. 鼠标悬浮在目标模板上，点击 **Apply** 按钮。
4. 为新添加的图表命名，并根据模板的物料层级选择具体的数据范围（Data Scope）。
5. 图表添加成功，显示在 Dashboard 中。

[截图: Use Template 弹窗，标注分组列表、模板卡片、Apply 按钮和底部提示文案]

**批量操作：**

在模板管理页面，你还可以进行以下批量操作：

- **Bulk Delete**：勾选多个模板后，点击 **Bulk Operation** → **Bulk Delete**，批量删除。
- **Bulk Move**：勾选多个模板后，点击 **Bulk Operation** → **Bulk Move**，选择目标分组，批量移动。
- **Bulk Share**（仅 HQ 场景）：勾选多个模板后，批量分享给其他 Client。

#### 8.8 通过模板快速创建 Dashboard

1. 在模板管理页面，勾选多个模板。
2. 点击 **Quick Create** 按钮。
3. 系统会校验所选模板的物料层级是否兼容：
   - 如果兼容，自动跳转到 Dashboard 编辑页面，已选模板对应的图表会自动添加到新 Dashboard 中。
   - 如果不兼容，弹窗提示具体原因，需要调整选择。

> **注意**：Quick Create 功能仅在 HQ 场景下可用，Commerce 场景暂不支持。

### 九、分享 Dashboard

#### 9.1 创建分享链接

1. 在 Dashboard 列表页，将鼠标悬浮到目标 Dashboard 上，出现 **分享** 图标。
2. 点击分享图标，系统自动生成分享链接并复制到剪贴板。
3. 弹出密码设置弹窗：
   - 如果之前未设置密码，弹窗提供设置密码的选项。
   - 如果已设置密码，弹窗显示当前密码，可修改或取消密码。

[截图: Dashboard 列表页悬浮状态的分享图标]

[截图: 分享链接密码设置弹窗]

#### 注意事项

- 只有正式保存（非草稿）的 Dashboard 才能生成分享链接。
- 分享链接是动态的——当你修改 Dashboard 内容后，通过链接看到的内容也会随之更新。

#### 9.2 设置分享密码

1. 在分享弹窗中，开启密码保护开关。
2. 输入访问密码。
3. 点击保存。

设置密码后，外部用户访问分享链接时需要先输入正确密码才能查看数据。

**修改或取消密码：**
- 再次点击分享图标，在弹窗中修改密码或关闭密码保护开关。

#### 9.3 锁定时间范围

**为什么需要锁定时间？**

默认情况下，分享链接中的数据时间范围会跟随 Dashboard 的设置变化。如果你在分享后修改了 Dashboard 的时间范围，外部用户看到的数据也会改变。锁定时间范围可以避免这个问题，确保外部用户只能看到你指定时间段的数据。

**操作步骤：**

1. 在分享设置弹窗中，开启 **锁定时间范围** 选项。
2. 设置锁定的起止日期。
3. 保存设置。

锁定后，外部用户访问分享链接时，时间选择器将被禁用，只能查看你指定时间段的数据。

#### 9.4 外部用户访问分享链接

外部用户（无需 Pacvue 账号）点击分享链接后：

1. 如果链接设置了密码，会弹出密码输入框：
   - 输入正确密码后进入 Dashboard 查看页面。
   - 密码错误会提示 "Password is wrong."，可重新输入。
2. 如果链接未设置密码，直接进入 Dashboard 查看页面。

## 常见问题

**Q: 保存为草稿和正式保存有什么区别？**

A: 草稿状态的 Dashboard 仅用于暂存你的配置进度，不会生成分享链接，也不会在分享页面中展示。正式保存后，Dashboard 才能被分享和正常使用。你可以随时将草稿正式保存。

---

**Q: 分享链接发出后，我修改了 Dashboard 内容，对方看到的会变吗？**

A: 会的。分享链接是动态链接，Dashboard 内容的修改会实时反映在分享页面上。如果你不希望对方看到修改后的内容，建议在分享前锁定时间范围，或在修改完成前暂时不分享。

---

**Q: 模板和图表有什么区别？**

A: 模板是一套预配置好的图表参数（如图表类型、指标选择、物料层级等），但不包含具体的数据范围。你可以把模板理解为"图表的配方"——引用模板后，还需要为图表指定具体的数据范围（如选择哪些广告活动），才能展示实际数据。模板的好处是可以复用，避免每次从头配置。

---

**Q: 为什么在 Use Template 弹窗中看不到某些模板？**

A: 系统会自动过滤掉与当前 Dashboard 数据源不兼容的模板。例如，如果你的 Dashboard 使用的是 Amazon 数据源，那么仅适用于 Commerce 的模板会被过滤掉。弹窗底部会显示提示信息，说明当前数据源和过滤原因。

---

**Q: 删除模板分组时，分组下的模板会怎样？**

A: 删除分组时，系统会提示你选择处理方式：可以将分组下的模板移动到其他分组，也可以连同模板一起删除。请根据实际需要选择。

---

**Q: 外部用户访问分享链接需要 Pacvue 账号吗？**

A: 不需要。分享链接支持免登录访问，外部用户只需要点击链接（如设置了密码则输入密码）即可查看 Dashboard 数据。外部用户只能查看，不能编辑任何内容。
